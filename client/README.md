## 🚀 كيفية الاستخدام

### التنقل في التطبيق

1. **الصفحة الرئيسية**: `/` أو `/home`
   - عرض الأغاني الشائعة والألبومات الجديدة
   - اكتشاف الفنانين المميزين
   - قوائم التشغيل المقترحة

2. **البحث**: `/search`
   - البحث عن أغاني، فنانين، وألبومات
   - تصفح حسب النوع الموسيقي
   - عمليات البحث الرائجة

3. **المكتبة**: `/library`
   - أغانيك المفضلة
   - ألبوماتك المحفوظة
   - قوائم التشغيل الخاصة بك

4. **صفحة الفنان**: `/artists/:id`
   - معلومات الفنان
   - أغاني الفنان
   - ألبومات الفنان

5. **لوحة الفنان**: `/artist/dashboard`
   - رفع أغاني جديدة
   - عرض الإحصائيات
   - إدارة المحتوى

6. **لوحة الإدارة**: `/admin/dashboard`
   - إدارة المستخدمين والفنانين
   - مراجعة المحتوى
   - معالجة البلاغات

### الوظائف الأساسية

#### تشغيل الموسيقى
1. انقر على أي أغنية لتشغيلها
2. استخدم شريط المشغل السفلي للتحكم
3. أضف أغاني إلى قائمة الانتظار

#### إدارة المكتبة
1. اضغط على أيقونة القلب للإعجاب بالأغنية
2. تابع فنانيك المفضلين
3. أنشئ قوائم تشغيل مخصصة

#### تبديل الثيم
1. اضغط على أيقونة الشمس/القمر في الشريط العلوي
2. التطبيق يدعم Dark Mode و Light Mode
